# Portfolio Personale — Morad Boumasdour

## Descrizione del progetto
Questo progetto consiste in un sito portfolio personale sviluppato per presentare il mio profilo, le competenze, i progetti, la formazione e le esperienze professionali. Il sito è stato realizzato con un design moderno, responsive e accessibile, pensato per essere facilmente navigabile sia da desktop che da dispositivi mobili.

## Funzionalità principali
- **Navigazione intuitiva**: tutte le sezioni principali sono accessibili tramite un menu in alto.
- **Cambio lingua dinamico**: è possibile alternare tra italiano e inglese, con traduzione automatica di titoli, contenuti e voci di menu.
- **Modalità chiara/scura**: toggle per passare tra light e dark mode, con colori e ombreggiature adattive.
- **Sezioni dedicate**: About Me, Skills, Formazione, Esperienze, Progetti, Contatti.
- **Form di contatto**: modulo centrale per invio messaggi.
- **Animazioni e overlay**: overlay di benvenuto animato all’apertura del sito.
- **Design responsive**: layout flessibile e ottimizzato per ogni schermo.

## Tecnologie utilizzate
- **HTML5** — Struttura semantica delle pagine
- **CSS3** — Stili, Flexbox, responsive design, dark mode, ombreggiature
- **JavaScript** — Interattività, gestione lingua, dark mode, overlay, aggiornamento dinamico dei contenuti
- **SVG** — Icone vettoriali per la navigazione e i link

## Struttura delle sezioni
- **About Me**: presentazione personale, contatti e lingue parlate
- **Skills**: elenco delle principali competenze trasversali
- **Formazione**: percorso di studi e certificazioni
- **Esperienze**: esperienze lavorative, di volontariato e progetti pratici
- **Progetti**: descrizione e link ai progetti principali su GitHub
- **Contatti**: form per invio diretto di messaggi e recapiti

## Modalità d’uso
1. Aprire il file `index.html` in un browser moderno.
2. Utilizzare il menu in alto per navigare tra le sezioni.
3. Usare i pulsanti in alto a destra per cambiare lingua o attivare la dark mode.
4. Compilare il form nella sezione Contatti per inviare un messaggio.
